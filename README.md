# Cryppy

A crypto price cheker for discord built with python

Built by:  ![](https://dcbadge.vercel.app/api/shield/208059872791494656?style=flat-square)

## Commands

Poll if the bot is running

    $ping
    
Check price of coin via its shortcode

    $price ETH
